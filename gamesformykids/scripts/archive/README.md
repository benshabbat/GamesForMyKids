# Archived Migration Scripts

These scripts were used for the one-time migration to Supabase and are no longer needed for active development.

| File | Purpose |
|---|---|
| `migrate-to-supabase.mjs` | Initial migration of game data to Supabase |
| `migrate-all-data.mjs` | Bulk data migration script |
| `complete-seed-migration.sql` | SQL seed data for completing the migration |

The migration is complete. These files are kept here for historical reference only.
